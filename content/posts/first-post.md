# Welcome

This is my first post on my Hugo portfolio website.

## Getting Started

Here's how you can set up your own Hugo site:

```bash
hugo new site mysite
cd mysite
git submodule add https://github.com/Livour/hugo-mana-theme.git themes/mana
```

Enjoy building with Hugo!
